# BLINKIT-DATA-ANALYSIS-USING-PYTHON
## Project Objective

This project delivers a comprehensive analysis of Blinkit product sales data using advanced Python data analytics techniques. The main goal is to uncover key distribution patterns, sales trends, and performance insights across various outlet types and item categories, enabling informed business decisions and optimization strategies
## Dataset Used
- <a href="https://github.com/vikaschennarapu/BLINKIT-DATA-ANALYSIS/blob/main/blinkit_data.csv">Dataset</a>

Questions (KPI's)
- Outlet Established Year and Sales According to the Year
- outlet location Based on Tier and Total sales
- Outlet Size
- Outlet Sales Based on Outlet Tier and Item Fat Content
- Product Sales Based by Item Fat Content
- Total Sales by Item Type

## Process
- Verify Data for any Missing Values and Anomalies, and Sort out the Same.
- Made Sure Data is Consistent and Clean with respect to Data Type, Data Format and Values Used.
- Explored and calculated all key metrics
- Visualized all patterns and trends
